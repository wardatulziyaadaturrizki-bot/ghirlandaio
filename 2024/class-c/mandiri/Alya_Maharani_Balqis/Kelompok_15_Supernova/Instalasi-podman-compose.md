## Mengecek SSH 
```
systemctl status sshd
```
Jika terdapat tulisan Active: active  (running) dan berwarna hijau tandanya sudah aktif

## Login ke server menggunakan SSH
```
ssh kel10@192.168.2.110 (no inetnya)
```
## menginstall Podman Compose
```
sudo pacman -S podman-compose
```
